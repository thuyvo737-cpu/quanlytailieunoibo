Chưa kết nối file dữ liệu
Trình duyệt hiện tại không hỗ trợ ghi trực tiếp vào file trên máy (cần Chrome hoặc Edge bản mới). Bạn vẫn thao tác bình thường — hệ thống sẽ tự tải một bản sao Excel về Downloads sau mỗi thay đổi.
Không tìm thấy sheet "Nguyên tắc đặt mã số" (hoặc thiếu bảng Tên đơn vị/Mã đơn vị) trong file này — mã MSTL sẽ được gợi ý theo cách suy đoán từ dữ liệu cũ thay vì theo đúng quy tắc chính thức.

Kết nối file dữ liệu trên máy

Chưa có file nào được liên kết.
Bấm nút bên dưới và chọn file Theo_dõi_cấp_số_tài_liệu_nội_bộ.xlsx đang có trên máy bạn. Sau khi liên kết, mọi thay đổi bạn làm trên trang này sẽ tự động ghi lại vào đúng file đó.
Dữ liệu không rời khỏi máy bạn — trang chỉ là mã HTML/JS tĩnh, host miễn phí (GitHub Pages / Cloudflare Pages). Đọc/ghi file qua tính năng của trình duyệt, không đi qua máy chủ nào.
